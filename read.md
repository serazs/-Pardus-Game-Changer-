# 🚀 Pardus Game Changer
Pardus'u oyun için optimize eden efsane araç.

## Kurulum
1. `git clone https://github.com/serazs/-Pardus-Game-Changer-.git`
2. `cd "./-Pardus-Game-Changer-"`
3. `sudo apt update && sudo apt install -y python3-pip libwebkit2gtk-4.1-dev python3-gi python3-gi-cairo gir1.2-gtk-3.0`
4. `pip3 install -r requirements.txt --break-system-packages`
5. `sudo python3 main.py`
